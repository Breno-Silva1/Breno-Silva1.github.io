Phase: Teste
A última interação é a fase de testes. O sistema deve estar completo e pronto para uso. A primeira fase de testes quem faz são os desenvolvedores e testadores, para verificar se todas os casos de usos estão devidamente incrementados e prontos para serem usados pelo cliente final. A segunda fase é o teste feito com o cliente. Com o sistema em mãos ele deverá usar o sistema sem dificuldades, caso isso aconteça, ajustes precisarão ser feitos.
